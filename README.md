# TowerDefence

[![Heroku](https://heroku-badge.herokuapp.com/?app=towerdefence)]

## Requirements

- node.js
- npm

## Install

```shell
git clone https://github.com/JosephAbbey/TowerDefence
cd TowerDefence
npm install
```

## Run

Dev:

```shell
npm start
```

Lint:

```shell
npm run lint
```

Build:

```shell
npm run build
```
